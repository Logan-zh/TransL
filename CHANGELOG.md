# 更新紀錄

本文件記錄 DEMOL 桌面版各版本的變更內容。

格式以 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/) 為參考，版本號遵循 [語意化版本](https://semver.org/lang/zh-TW/)。

## [Unreleased]

## [4.1.3] - 2026-06-16

### 修正

- 開機自動啟動時：已登入使用者不再自動彈出登入視窗；僅在背景執行於系統匣
- 開機啟動時若需登入，改以系統匣提示，可從右鍵選單開啟登入

---

### 修正

- Windows 桌面捷徑：安裝時移除舊版 TransL 捷徑，啟動時自動刪除殘留的 `TransL.lnk`
- 設定 `AppUserModelId`（`com.demol.app`），讓工作列／捷徑顯示 DEMOL 白底 D 圖示

---

### 新增

- Vitest 單元測試：API 連線錯誤、下載 URL 解析、IPC 錯誤訊息、SEO 區塊讀取

---

## [4.1.0] - 2026-06-16

### 變更

- **UI 與前台統一**：淺色簡潔風格（`#fafafa` 背景、`#111827` 主色），登入、設定、翻譯浮動窗、截圖選區、選取小圖示全面更新
- **圖示**：白底「D」應用程式與系統匣圖示

---

## [4.0.0] - 2026-06-02

### 變更

- **產品更名 DEMOL**：桌面版顯示名稱、系統匣、通知、登入／設定視窗、安裝捷徑與 `appId` 全面更新
- 共用常數 `APP_NAME` 移至 `@transl/shared`，主程序與渲染程序共用

---

## [3.0.4] - 2026-06-12

### 變更

- **Ctrl+Q 回覆建議**：【理解】固定繁中摘要；各建議回覆附繁中語氣／用途說明

---

## [3.0.3] - 2026-06-12

### 修正

- **翻譯彈窗定位**：選取文字在畫面中下方時，下方空間不足改為向上平移貼齊工作區底部，不再整窗翻到游標上方而偏高

---

## [3.0.2] - 2026-06-07

### 修正

- **翻譯彈窗定位**：觸發當下固定錨點（滑鼠／選取圖示／截圖區右下角），同一次翻譯不再重複 reposition；修正 Y 軸邊界計算錯誤導致視窗被系統推擠亂跳

---

## [3.0.1] - 2026-06-07

### 修正

- **雙擊 Ctrl+C 翻譯**：改為偵測鍵盤「0.8 秒內連按兩次 Ctrl+C」，不再依剪貼簿序號連續變動觸發；修正 Ctrl+A 後單次 Ctrl+C 就誤開翻譯、燒掉額度的問題
- 觸發前仍檢查剪貼簿須有文字（僅圖片不觸發）；**Ctrl+Alt+雙擊 S** 截圖翻譯不受影響

---

## [3.0.0] - 2026-06-07

### 變更

- **架構重構**：新增 monorepo 共用套件 `@transl/shared`（翻譯型別與 prompt、語言偵測、HTTP client、會員／配額型別、格式化工具）
- 桌面端、後端 API、public-web、admin-web 改從 shared 匯入，消除重複定義與漂移風險
- 桌面端內部整理：型別與 IPC／快捷鍵常數集中、移除 2.0 後未使用的本機 LLM provider 死碼、Win32 FFI 統一至 `win32.ts`
- 後端 Docker 建置改從 monorepo 根目錄編譯 `@transl/shared`

### 新增

- 翻譯彈窗 **修改** 選單「翻譯為」**繁中** 選項；選項過多時可滾動

---

## [2.6.1] - 2026-06-05

### 新增

- 翻譯彈窗 **修改** 選單可指定譯文語言：**英文**、**韓文**、**日文**（自動辨識原文語系後重譯）

---

## [2.6.0] - 2026-06-02

### 變更

- **自動語言方向**：含中文的選取文字翻譯為英文；其餘語言（英文、韓文、日文等）自動譯為繁體中文（不再假設非中文即英文）
- 截圖翻譯與語氣調整同步套用上述規則
- **發音**：依原文粗判語系（繁中／韓文／日文／英文等），使用對應系統語音

## [2.5.0] - 2026-06-02

## [2.0.0] - 2026-05-31

### 新增

- **會員登入版**：桌面端改為以管理後台建立的帳號登入
- 新增登入視窗與 API 伺服器位址設定
- 翻譯請求改由 **demol-admin 後端代理**，不再於本機儲存 API Key

- 1.x 本機 API Key 設定不再使用（偵測到時顯示遷移提示）

### 移除

- 設定頁的 OpenAI / Gemini API Key 與模型選項

- 透過 Windows `GetAsyncKeyState` 偵測 Ctrl+D 雙擊（0.8 秒內）

### 變更

## [1.1.0] - 2026-05-31

### 新增

- 翻譯浮動窗新增 **貼上** 按鈕：將譯文貼回原本應用程式
- 翻譯浮動窗新增 **修改** 按鈕，可選擇：
  - **更平易近人**：口語、自然、好懂的譯文
  - **更專業**：正式、商務或學術語氣的譯文
- 設定頁新增 **開機時自動啟動** 選項

### 變更

- 調整語氣重新翻譯時，浮動窗 **保留在原位置**，不再跟著游標移動
- 貼上流程改為先還原目標視窗焦點，再模擬 `Ctrl+V`，提升成功率

### 修復

- 修正打包時 `icon.ico` 格式錯誤導致 `electron-builder` 失敗的問題
- 修正早期鍵盤監聽套件無法正常運作，改以 Windows 剪貼簿序號偵測雙擊 `Ctrl+C`

## [1.0.0] - 2026-05-31

### 新增

- Windows 桌面版 TransL 首次發布
- 選取文字後，**0.8 秒內連按兩次 `Ctrl+C`** 觸發翻譯
- 英文 → 繁體中文；中文 → 英文（自動偵測）
- 支援 **OpenAI** 與 **Google Gemini** API
- Always-on-top **浮動翻譯窗**，顯示於游標附近
- **系統匣**常駐，提供設定、重新載入監聽、翻譯剪貼簿、結束
- 設定視窗：Provider、API Key、Model
- Windows 安裝程式打包（NSIS）
