# 更新紀錄

本文件記錄 TransL 各版本的變更內容。

格式以 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/) 為參考，版本號遵循 [語意化版本](https://semver.org/lang/zh-TW/)。

## [2.0.0] - 2026-05-31

### 新增

- **會員登入版**：桌面端改為以管理後台建立的帳號登入
- 新增登入視窗與 API 伺服器位址設定
- 翻譯請求改由 **TransL-admin 後端代理**，不再於本機儲存 API Key

### 變更

- 設定頁改顯示登入狀態、指派之翻譯服務與登出
- API 伺服器位址改由環境變數 `TRANSL_API_URL` 設定（建置／部署時配置，使用者不可修改）
- 1.x 本機 API Key 設定不再使用（偵測到時顯示遷移提示）

### 移除

- 設定頁的 OpenAI / Gemini API Key 與模型選項

---

## [1.3.2] - 2026-05-31

### 新增

- 浮動窗譯文圖預覽支援 **滾輪縮放**、**拖曳移動**，以及 +/- / 重設按鈕

### 修復

- 修正拖曳譯文圖時觸發瀏覽器原生 **拖移圖片** 或外層捲軸的問題；圖片 viewer 獨立於文字捲動區

---

## [1.3.1] - 2026-05-31

### 新增

- 截圖翻譯：本地將譯文 **疊加至原圖**（方案 A），剪貼簿改存 **譯文圖**
- Vision API 回傳文字區塊座標，以白底遮蓋原文字後繪製譯文

### 變更

- 浮動窗截圖預覽改顯示 **譯文疊圖** 結果
- 調整語氣時會重新合成譯文圖並更新剪貼簿

---

## [1.3.0] - 2026-05-31

### 新增

- **Ctrl + Alt + 雙擊 S**：全螢幕框選截圖，以 Vision API 辨識圖中文字並翻譯
- 截圖完成後 **自動寫入剪貼簿**（可 Ctrl+V 貼上圖片）
- 浮動窗顯示截圖縮圖、辨識原文與譯文，支援貼上與修改語氣

### 變更

- 啟動提示與系統匣 tooltip 更新為三組快捷鍵說明

---

## [1.2.2] - 2026-05-31

### 變更

- Ctrl 雙擊 C 浮動窗：原文與譯文 **保留換行與段落排版**，方便逐行閱讀對照
- 翻譯 prompt 要求 AI 維持原文的行距與空行結構

---

## [1.2.1] - 2026-05-31

### 修復

- 修正 Ctrl 雙擊 D 翻譯貼上時，因程式寫入剪貼簿而誤觸 Ctrl 雙擊 C 浮動窗的問題

---

## [1.2.0] - 2026-05-31

### 新增

- **Ctrl 雙擊 D**：框選文字後直接翻譯並貼上，不顯示浮動窗
- 透過 Windows `GetAsyncKeyState` 偵測 Ctrl+D 雙擊（0.8 秒內）

### 變更

- 啟動提示與系統匣 tooltip 更新為雙快捷鍵說明

---

## [1.1.0] - 2026-05-31

### 新增

- 翻譯浮動窗新增 **貼上** 按鈕：將譯文貼回原本應用程式
- 翻譯浮動窗新增 **修改** 按鈕，可選擇：
  - **更平易近人**：口語、自然、好懂的譯文
  - **更專業**：正式、商務或學術語氣的譯文
- 設定頁新增 **開機時自動啟動** 選項

### 變更

- 調整語氣重新翻譯時，浮動窗 **保留在原位置**，不再跟著游標移動
- 貼上流程改為先還原目標視窗焦點，再模擬 `Ctrl+V`，提升成功率

### 修復

- 修正打包時 `icon.ico` 格式錯誤導致 `electron-builder` 失敗的問題
- 修正早期鍵盤監聽套件無法正常運作，改以 Windows 剪貼簿序號偵測雙擊 `Ctrl+C`

---

## [1.0.0] - 2026-05-31

### 新增

- Windows 桌面版 TransL 首次發布
- 選取文字後，**0.8 秒內連按兩次 `Ctrl+C`** 觸發翻譯
- 英文 → 繁體中文；中文 → 英文（自動偵測）
- 支援 **OpenAI** 與 **Google Gemini** API
- Always-on-top **浮動翻譯窗**，顯示於游標附近
- **系統匣**常駐，提供設定、重新載入監聽、翻譯剪貼簿、結束
- 設定視窗：Provider、API Key、Model
- Windows 安裝程式打包（NSIS）
